Map Menu Items


Map options

Selecting Map>Map options... from the main menu opens a dialogue such as the one below.

You can set "map lifetime", i.e. the length ("n" seconds) of the vessel track line here, how long data to be displayed on the map should be stored for and how long the data should be displayed for.

Check the 'show hydrophones' box to display the positions of the array elements on the map

Check the 'Keep ship on map' box to automatically keep the ship on the map by re-centering it whenever the ship moves beyond the edge of the display.





Map file

Use the browse button to select a map file. PAMGUARD currently only supports ASCII text files in the format exported by the 2000 version of the Gebco digital atlas. Instructions on preparing these can be found here...

When a map file is opened a series of check boxes will appear enabling the user to select the contour lines they wish to display on the map.
