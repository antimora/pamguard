Rocca

Overview


The ROCCA (Real-time Odontocete Call Classification Algorithm) module classifies delphinid whistles based on spectrographic measurements extracted from whistle contours. The contour extraction algorithm measures 54 different variables for each contour. A subset of these variables is then used in conjunction with a user-selected classification model to classify the whistle to species.


Individual whistle classifications are grouped into user-defined Detection Numbers. A classification is also given to detection numbers based on the species within that group with the highest classification scores.


General Principle of Operation

There are 5 main steps to the classification of a whistle:

  1. User selects a whistle from the spectrogram display

  2. The whistle is captured and displayed in a new spectrogram window

  3. The whistle contour is extracted and the user is allowed to manipulate it, if desired

  4. The contour parameters are measured

  5. The contour is classified using the currently-loaded classifier model

  6. The classified whistle statistics are added to the specified School/Detection/Sighting
