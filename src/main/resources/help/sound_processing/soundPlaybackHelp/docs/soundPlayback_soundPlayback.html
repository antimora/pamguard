Sound Output

Overview

The sound output module can be used to play back sounds from PAMGUARD through a sound card or other data acquisition device.

This allows you to aurally monitor sounds after they have been modified by PAMGUARD modules such as IIRF Filters or the Seismic Veto

The availability of the sound output function depends on the type of input selected in the Sound Acquisition module.

With the exception of data read from file, sound output must be through the same device as is being used for sound input so that there is perfect synchronisation between the clocks of the device being used for output and the device used for input.

The sound output module will currently work when analysing data from files of file folders, from sound cards, ASIO sound cards and with simulated data. It will not currently work with National Instruments DAQ cards.


Creating sound output modules

From the File>Add modules>Sound Processing menu, or from the pop-up menu on the data model display select "Sound Output". Enter a name for the new module and press Ok.

Sound output modules can be inserted anywhere in the PAMGUARD data model where there are raw audio data such as the output of a sound acquisition module, a filter module, a decimator, etc.
